Smart Waste Management System using ESP32

 Project Description

This project is a smart garbage monitoring system using ESP32 microcontroller. It detects the garbage level in bins and sends alerts when the bin is full.

 Objectives

- Monitor garbage level in real-time
- Avoid overflow of waste bins
- Improve waste collection efficiency

 Components Used

- ESP32 Microcontroller
- Ultrasonic Sensor (HC-SR04)
- WiFi Module (inbuilt)
- Buzzer / LED
- Power Supply

 Working

The ultrasonic sensor measures the garbage level. ESP32 processes the data and sends alerts using WiFi when the bin is full.
